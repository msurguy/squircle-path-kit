# Product

## Register

product

## Users

Designers and engineers who need to generate Figma-like smoothed corners for rectangles, icons, buttons, image masks, and arbitrary SVG shapes. They are usually testing a visual shape, copying implementation code, or verifying that output matches Figma closely enough for production UI work.

## Product Purpose

Squircle Path Kit generates SVG path data for closed polygons with Figma-style corner radius and corner smoothing. Success means users can quickly inspect visual examples, tune parameters, copy exact code for their target platform, and trust that the library remains tiny and framework-independent.

## Brand Personality

Playful creative, precise, visual. The interface should make geometry feel approachable without hiding the math or turning the tool into a marketing page.

## Anti-references

Avoid generic SaaS landing-page layouts, heavy decorative gradients, oversized hero sections, and docs pages that make users scroll before trying the tool. Avoid visual treatments that obscure the generated shape or make code examples hard to scan.

## Design Principles

- Put the working playground first.
- Show code that exactly matches the visible result.
- Keep the library invisible: no framework assumptions, no DOM dependency, no runtime dependencies.
- Make control geometry inspectable without making the default view noisy.
- Use playful color in examples, not at the expense of readability.

## Accessibility & Inclusion

Target WCAG AA. The site needs visible keyboard focus, sufficient contrast, semantic controls, copyable code without pointer-only flows, and reduced-motion support for nonessential animation.
